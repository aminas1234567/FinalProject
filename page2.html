Web Technologies

Frontend Technologies (visual aspects that users see)

Technology Description Example
HTML Hypertext Markup Language used for structuring web pages. <html><body></body></html>
CSS Cascading Style Sheets used for styling web pages. body { color: red; }
JavaScript Programming language used for interactive elements on web pages. alert('Hello, World!');

Backend Technologies (data and infrastructure that make an application work)

Technology Description Example
PHP Server-side scripting language used for dynamic web pages. <?php echo 'Hello, World!'; ?>
Node.js JavaScript runtime environment used for server-side programming. console.log('Hello, World!');
Python General-purpose programming language often used for web development. print('Hello, World!')

Source: GfG. “Frontend vs Backend.” GeeksforGeeks, GeeksforGeeks, 18 Apr. 2023, www.geeksforgeeks.org/frontend-vs-backend/.
